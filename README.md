# 🏗️ AGR XML Generator

WPF-приложение для формирования XML-файлов архитектурно-градостроительных решений (АГР) в соответствии с утвержденной XML-схемой.
https://www.mos.ru/upload/content/files/7fc24614064f95e83de48ed566e27614/RDGPiDITpoIFC.pdf

## 📋 Возможности

✅ Заполнение данных объекта строительства

✅ Ввод кадастровых номеров, ГПЗУ, ППТ, ППМ и других документов

✅ Формирование состава авторского коллектива

✅ Указание функционального назначения объекта

✅ Проверка обязательных полей

✅ Контроль корректности числовых значений

✅ Генерация XML-файла в требуемом формате

✅ Загрузка тестового примера для проверки

✅ Очистка формы одним нажатием

---

## 🖥️ Интерфейс

Программа позволяет заполнить:

- Наименование объекта
- Регистрационный номер
- Дату
- Адрес
- UUID
- Проектную организацию
- Руководителя авторского коллектива
- Вид работ
- Кадастровые номера
- ГПЗУ
- ППТ
- Решения ГЗК
- КРТ
- ППМ
- Состав авторского коллектива
- Функциональное назначение объекта

а также основные технико-экономические показатели:

- Площадь участка
- Общая площадь
- Этажность
- Высота объекта
- Количество квартир
- Озеленение
- Площадки отдыха
- Детские площадки
- Информационные конструкции

## 📄 Формируемый XML

При сохранении создается XML-файл вида:

```xml
<ArchitecturalUrbanPlanningSolution>
    <ObjectName>...</ObjectName>
    <Address>...</Address>
    <ProjectOrganization>...</ProjectOrganization>
    ...
</ArchitecturalUrbanPlanningSolution>

## Сборка

```powershell
dotnet publish -c Release -r win-x64 --self-contained true /p:PublishSingleFile=true
```

Готовый EXE:

```text
bin\Release\net8.0-windows\win-x64\publish\
```
